# Year 1 Unit 1 Introduction to Web Development
Guided Learning Materials and Assignment Templates for Sentinel Workshop Year 1 Introduction to Web Development

## Topics
- HTML Basics
- Styles
- JS Basics
- Adding Interaction
- Conditionals

## HTML Basics
### Brief Summary
HTML is used to create the structure and content of a webpage.  
In this topic, students will learn how to use common HTML elements such as headings, paragraphs, lists, links, and images to build a simple webpage.
### Materials Provided
- HTML Template file
- Hello HTML Assignment guided solution (Went Through in Class)
- My Favorite Things Assignment guided solution (Went Through in Class)

## Styles
### Brief Summary
Styles are used to change how a webpage looks.  
In this topic, students will learn how to use CSS to change colours, fonts, sizes, spacing, and layout to make their webpages more visually appealing.
### Materials Provided
- Color Me Interested Assignment guided solution (Went Through in Class)
- Tom and Jerry Assignment guided solution (Went Through in Class)

## JS Basics
### Brief Summary
JavaScript is used to make webpages more dynamic.  
In this topic, students will learn basic JavaScript concepts such as displaying messages, using variables, and writing simple code that runs in the browser.
### Materials Provided
- Hello World Assignment guided solution (Went Through in Class)
- Global Clock Assignment guided solution (Went Through in Class)

## Adding Interaction
### Brief Summary
Interaction allows users to do something on a webpage and see a response.  
In this topic, students will learn how to use JavaScript with buttons, input fields, and events to make webpages respond to user actions.
### Materials Provided
- Gotta Get You Assignment guided solution (Went Through in Class)
- Favorite Color Assignment guided solution (Went Through in Class)

## Conditionals
### Brief Summary
Conditionals allow a program to make decisions.  
In this topic, students will learn how to use `if`, `else if`, and `else` statements to check conditions and control what happens based on user input.
### Materials Provided
- Let Me In 1 Assignment guided solution (Went Through in Class)
- Let Me In 2 Template file
- Let Me In 3 Assignment guided solution (Went Through in Class)
